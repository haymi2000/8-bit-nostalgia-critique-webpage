# 8-Bit Nostalgia Critique Webpage

## Project Overview (Lecture-Related)
This project was developed as part of a coursework assignment focused on **Web Design Principles** and **Retro UI Aesthetics**. The goal of the project is to critique and implement design elements inspired by the 8-bit video game era using modern web technologies.

##  Key Features
* **Retro Critique:** Analysis of 8-bit visual styles applied to modern web layouts.
* **Responsive Design:** Optimized for both mobile and desktop viewing.
* **CSS Implementation:** Custom styling to mimic pixel-art and nostalgic color palettes.

## Tech Stack
* **HTML5** - For semantic web structure.
* **CSS** - For layout and retro-inspired styling.
* **GitHub Pages** - Used for live deployment and hosting.
